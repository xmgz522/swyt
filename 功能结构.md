# 三位一体测评系统 — 完整功能结构

## 技术架构
- 后端：NestJS + MySQL + TypeORM
- 后台管理：Vue3 + Element Plus
- 学生PC端：Vue3 + Element Plus + TailwindCSS
- 小程序端：uni-app（微信小程序 / H5）

---

## 一、后台管理系统（admin）

### 1. 数据总览 /dashboard
- 学生总数、本周新增学生数
- 最近7天每日新注册趋势
- 试卷总数、已发布试卷数
- 题库总题数
- 待批改答卷数

### 2. 学生管理 /students
- 学生列表（姓名、手机、学校、年级、学考成绩、注册时间）
- 搜索筛选
- 查看/编辑学生详情
- 删除学生

### 3. 院校管理 /schools（管理员+）
- 院校列表（名称、层次、省份、类型、三位一体招生信息）
- 新增/编辑/删除院校
- 院校详情：招生计划、分数线、报考条件等

### 4. 考试中心
#### 4.1 题库管理 /questions
- 题目列表（类型、科目、知识点、难度、分值、所属题库）
- 支持单选/多选/判断/填空/简答等题型
- 新增/编辑/删除题目
- 按科目、知识点、题库类型（triad三位一体 / xuekao学考）筛选
- 批量导入题目

#### 4.2 试卷管理 /papers
- 试卷列表（名称、科目、总分、时长、题目数、发布状态、题库类型）
- 创建/编辑试卷（选择题目组卷）
- 发布/取消发布
- 删除试卷

#### 4.3 批卷管理 /grading
- 待批改答卷列表
- 主观题逐题打分
- 提交批改（自动计算总分、更新答卷状态）

#### 4.4 答卷记录 /answer-sheets
- 所有学生答卷记录
- 查看答卷详情（每题作答与评分）

#### 4.5 面试题库 /interview
- 面试题列表（题目、类别、难度、参考答案、答题要点）
- 新增/编辑/删除面试题
- 按类别筛选

#### 4.6 模拟面试 /mock-interview
- 模拟面试场次管理
- 查看面试记录

### 5. 数据分析
#### 5.1 错题管理 /wrong-notes
- 全部学生错题汇总
- 按学生、科目、掌握状态筛选
- 错题统计

#### 5.2 AI 分析 /analysis
- 学生薄弱项分析结果查看
- 按学生查看各科正确率、薄弱知识点

### 6. 测评报告 /report
- 学生测评报告列表
- 查看/生成测评报告

### 7. 学情周报 /weekly-report
- 按周查看周报汇总（人数、平均正确率、平均做题量）
- 手动「生成本周周报」按钮
- 每周一 9:00 自动生成
- 查看某周所有学生周报详情
- 推送周报通知给学生

### 8. 专业管理 /majors
- 专业列表（名称、代码、学科门类、学制、学位、状态）
- 新增/编辑/删除专业
- 搜索、按学科门类筛选
- 一键导入预置专业数据（20个常见专业）

### 9. 政策管理 /policies（管理员+）
- 政策/公告列表
- 新增/编辑/删除政策
- 发布/下架

### 10. 消息通知 /notifications
- 通知列表
- 发送通知给指定学生
- 删除通知

### 11. 导入导出 /import-export
- 学生数据导出
- 题库批量导入
- 院校数据导入

### 12. 系统设置
#### 12.1 用户管理 /users（超管）
- 后台用户列表（管理员、教师）
- 新增/编辑/删除用户
- 角色分配（superadmin / admin / teacher）

#### 12.2 操作日志 /audit-logs（超管）
- 后台操作记录追溯

#### 12.3 修改密码 /change-password
- 当前用户修改密码

#### 12.4 系统配置 /config（超管）
- 系统参数配置

## 二、学生PC端（student-web）

### 首页 /
- 个人学考成绩概览（几A几B）
- 功能入口网格（12个快捷入口）
- 最近考试记录
- 今日学习任务
- 目标院校倒计时

### 院校推荐 /recommend
- 填写/更新学考成绩
- 智能推荐院校（冲刺/稳妥/保底三档）
- 院校详情页（招生信息、分数线、报考条件）

### 模拟测试 /exams
- 可用试卷列表
- 在线答题（计时、自动交卷）
- 成绩报告（得分、正确率、每题解析）

### 历年真题 /exam-real
- 历年真题试卷列表
- 在线做题

### 学考冲刺 /sprint
- 学考专属模块
- 学考薄弱项分析（按科目）
- 学考专项练题（按知识点分类）
- 学考错题本
- 学考试卷列表

### 考点练题 /practice
- 按科目、知识点分类练习
- 智能出题

### 薄弱项分析 /analysis
- 各科正确率统计
- 薄弱知识点排名
- 学习建议

### 我的错题 /wrong-notes
- 错题统计（待攻克/已掌握/总数）
- 按科目分布
- 错题列表（展开查看解析）
- 标记掌握/取消掌握
- 按题库类型、科目、掌握状态筛选

### 面试题库 /interview
- 面试题分类浏览
- 查看参考答案和答题要点

### 模拟面试 /mock-interview
- 模拟面试练习
- 计时答题

### 学情周报 /weekly-report
- 历史周报列表
- 周报详情（做题量、正确率、打卡天数、薄弱点、与上周对比）

### 专业解析 /majors
- 专业列表（搜索、按学科门类筛选）
- 专业详情弹窗（简介、核心课程、就业方向、适合人群、相关院校）

### 学习日历 /calendar
- 每日学习任务
- 任务完成打卡

### 政策中心 /policies
- 最新政策/公告浏览

### 消息通知 /notifications
- 通知列表（未读红点）
- 标记已读/全部已读
- 删除/清空通知

### 我的信息 /profile
- 个人资料查看/编辑
- 学考成绩管理

|------|---------|------|
| AuthModule | /auth | 登录注册、JWT 鉴权 |
| StudentModule | /students | 学生 CRUD、资料管理 |
| SchoolModule | /schools | 院校 CRUD、详情 |
| RecommendModule | /recommend | 院校智能推荐算法 |
| ExamModule | /exam | 题库、试卷、答卷、批改、错题、练习、分析 |
| InterviewModule | /interview | 面试题库、模拟面试 |
| MajorModule | /majors | 专业 CRUD、预置数据 |
| ReportModule | /report | 测评报告、学情周报 |
| StudyPlanModule | /study-plan | 学习日历、任务打卡 |
| PolicyModule | /policies | 政策公告 |
| NotificationModule | /notifications | 消息通知、未读计数 |
| DashboardModule | /dashboard | 后台数据总览统计 |
| ExportModule | /export | 数据导入导出 |
| AuditModule | /audit | 操作日志 |
| ConfigModule | /config | 系统配置 |
